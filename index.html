Théorie Mathématique et Géométrique — L’Univers est au Carré

Bienvenue sur la page officielle du dépôt GitHub consacré à la théorie L’Univers est au Carré, développée par Philippe Thomas Savard.

Ce dépôt public, sous licence Apache 2.0, constitue un espace ouvert permettant à tous de consulter, étudier, modifier, cloner ou contribuer à cette théorie mathématique et géométrique innovante.

Documentation complète (PDF LaTeX certifiés)

Le dépôt contient plus d’une dizaine de documents PDF générés à partir de sources LaTeX et certifiés automatiquement via GitHub Actions. Ces documents exposent en profondeur les concepts de la théorie.

  • Documents mathématiques détaillés
  • Constructions géométriques
  • Analyses spectrales
  • Développements arithmétiques
  • Validations formelles Isabelle/HOL intégrées

Note : seul le document spirale_inverse_temps.pdf ne contient pas de validation HOL.

Validations Isabelle/HOL (≈ 4000 lignes)

La théorie est accompagnée de 8 validations formelles Isabelle/HOL, représentant environ 4000 lignes de code.

Ces validations assurent une rigueur mathématique aux concepts présentés, couvrant :

  • la géométrie du spectre des nombres premiers
  • la méthode spectrale
  • les fondements arithmétiques
  • les structures internes de la théorie

Animation YouTube (1h20) — Script narratif inclus

Une animation vidéo d’environ 1h20 présente la théorie sous forme narrée. Le dépôt inclut :

  • le script narratif complet
  • les illustrations
  • les éléments utilisés dans l’animation

Évaluation professionnelle (score : 74.3 / 100)

Une évaluation professionnelle complète est générée automatiquement via GitHub Actions, basée sur :

  • Greiffenhagen (2023)
  • K-State Proof Rubric
  • Calgary Peer-Proof Rubric
  • Processus CRM

L’auteur a obtenu un score global de 74.3 / 100.

À noter : 85 % de la théorie repose sur la méthode spectrale et la géométrie du spectre des nombres premiers, pour lesquelles l’auteur a obtenu 19 / 20.

Mot de l’auteur

Bien que l’auteur ne soit pas issu du milieu académique des mathématiques, ce travail représente un projet personnel sérieux, structuré et passionné, né d’un intérêt profond pour les nombres premiers et les constructions du type :

n^(1/2) + un entier

L’objectif n’est pas de créer un mythe ou une aura ésotérique, mais simplement de partager un travail personnel ouvert, perfectible et accessible à tous.
