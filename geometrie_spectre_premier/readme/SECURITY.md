*# Politique de sécurité

## Versions prises en charge

Ce dépôt est un projet scientifique versionné de manière continue.  
La seule version officiellement supportée est **la version la plus récente** publiée sur la branche `main`.

| Version | Support |
|--------|---------|
| main   | ✔ Supportée |
| autres branches | ✖ Non supportées |

Les versions modifiées par des tiers (forks, branches personnelles, contributions externes) ne sont **pas** supportées.

## Signalement d’une vulnérabilité

Si vous pensez avoir identifié une faille de sécurité dans ce dépôt :

1. **Ne créez pas d’issue publique.**
2. Contactez l’auteur principal en privé :  
   **philippe.savard (adresse email)**  
3. Décrivez clairement :
   - la nature de la vulnérabilité,
   - les fichiers concernés,
   - les étapes pour la reproduire (si possible).

Vous recevrez une réponse dans un délai raisonnable, selon la disponibilité de l’auteur.
